Essa biblioteca possui como função gerenciar e instalar as funções de Python nas máquinas, facilitando a gestão de cada projeto e suas peculiaridades.

Download WIN: https://github.com/pyenv-win/pyenv-win
Download  LINUX: https://github.com/pyenv/pyenv

Comandos:

pyenv --version -> Saber a versão da lib

pyenv versions -> Saber as versões de Python que a lib gerencia 

pyenv install 3.12.5 -> Instalar uma versão de Python

pyenv global 3.12.5 -> Estabelece uma versão global, ou seja, caso não estabeleça uma versão, será utilizado está.

pyenv uninstall 3.12.5 -> Exclui a versão do python 

pyenv local 3.12.5 -> informa para a lib que está pasta precisa rodar determinada versão
